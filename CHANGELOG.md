# Changelog

All notable changes to the VS-Leet extension will be documented in this file.

## [1.0.3] - 2025-12-19

### Changed
- Extension now activates only when clicking the activity bar icon (lazy loading)
- Improved logo design with better visibility at small sizes

### Fixed
- Login flow stability improvements
- Activity bar icon rendering issues
- Missing command definitions

## [1.0.2] - 2025-12-15

### Added
- Enhanced problem filtering capabilities
- Better error handling for submission flow

### Fixed
- Bug fixes and performance optimizations

## [1.0.1] - 2025-12-10

### Fixed
- Minor bug fixes
- Improved error messages

## [1.0.0] - 2025-12-01

### Added
- Initial release of VS-Leet
- Browse LeetCode problems from activity bar
- Filter by difficulty and tags
- Run test cases directly in VS Code
- Submit solutions to LeetCode
- Persistent login with Chromium automation
- Multi-language support (C++, Java, Python, JavaScript, TypeScript, and more)
- Automatic solution file management
